Options
User Interface
Base Font:
Sets the font used for all screens and menus.
Show the Userid in the Tree:
Shows the userid as part of the title in the tree screen.
Hide Title Bar:
Hides the title bar at the top of all screens.
Hide Icons:
Hides the icons in the tree view.
Hide Entry URL, Hide Entry Note Title:
Saves space on the entry screen.
Prompt on Delete:
Prompts before deleting entries or groups.
Search Mode:
Title starts with:
Finds entries with titles starting with the search criteria.
Any field contains:
Finds entries where the search criteria is contained in any of the entry's fields.
Find As You Type:
Allows you to search by just typing character in the tree screen. Otherwise you have to use the "Find" menu item.
Expand Groups on Startup:
Automatically expands all groups in the tree when KeePassBB is started.
Auto Confirm Group/Entry Changes:
Automatically confirms group/entry changes.
Security
Inactivity Timeout:
If no activity is detected in this interval, KeePassBB will be closed.
Escape Hides KeePassBB:
If you press the escape key at the main KeePassBB screen, the app will go into the background instead of closing. The database remains open until the Inactivity Timeout expires. Reopening KeePassBB before the timeout will bring you back to an open database.
Close When Holstered:
Closes KeePassBB when the device is placed in a holster with a sleep magnet.
Restricted Mode:
Allows access to only a few menu options unless a database is open. Options is not available in restricted mode.
Remember Last Used Directories:
Will remember last used directories when opening or saving database or key files.
Database
Read Only:
Just that. No database changes will be allowed.
Mark DB Modified on Last Access Changes:
If set, viewing a group or entry will cause the database to require saving.
Auto save every change:
Every change to an entry or group will cause an immediate database save. If you have a large database, this might not be a good idea but it is the safest option.
Auto save on app close:
When you close the application, the database will automatically be saved.
Auto save on app hide:
When you hide the application, the database will automatically be saved.
Reload database on Start:
When you start KeePassBB, your database will automatically be loaded.
Use External File Mode:
See the FAQ for more info.
Specify a database file for Reload Database on Start:
External Mode Only: Specify the database you wish to auto open. You also can specify an HTTP URL.
Disable MDS proxy for HTTP requests...:
When you open a URL and you're connected to a BlackBerry Enterprise server, the default is to route your request through the BES/MDS. If you experience problems opening URLs, you can force direct TCP connections by disableing the MDS proxy.
KeyFile
Hide KeyFile on Password Dialog:
Hides the keyfile information on the password dialog box.
Load KeyFile on Startup:
Automatically loads the keyfile specified below on startup.
**Note**Using both keyfile options gives you the ability to open a keyfile protected database without letting anyone else know that a keyfile is even needed.
